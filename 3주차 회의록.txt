같은 위치에 있는 계단이나 에스컬레이터는 노드를 나누지 않는다.
계단이나 엘리베이터 같은 이동 수단과 출구는 강의실 같은 하위 노드 취급한다.
노드 사이에 계단이나 엘리베이터가 있다면 노드끼리도 잇고 계단도 이동 수단도 그 두 노드에 잇는다.

이동 수단은 최단 경로의 끝이 아닌 중간에 위치하므로
이동 수단은 하위 노드가 아닌 노드로 한다.

그래프 구조 설정

그래프 클래스{
	플로어 클래스{
		노드 클래스{
		}
	}
}

>>노드 클래스는 교차점/엘리베이터/계단/에스컬레이터 중 하나이다.
char형의 distinction 필드로 구분한다. N이면 교차점, E이면 엘리베이터, S이면 계단, C이면 에스컬레이터
>>노드 사이의 거리는 dictionary 자료형으로 받아서 인접 노드의 번호를 key, 거리를 value로 한다.
인접 노드는 추가가 용이한 ArrayList로 받는다.
>>인접 강의실은 강의실 이름과 거리를 각각의 배열로 필드 선언하여
인덱스를 일치시켜 찾는다.
>>플로어/그래프 클래스는 각각 헤드와 테일 노드/플로어를 필드로 갖는다.
